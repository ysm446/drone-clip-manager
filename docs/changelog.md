# Changelog

このファイルはユーザー向けの変更履歴。日時は `YYYY-MM-DD HH:MM` 形式。

## 未リリース

### 2026-07-08 23:35
- **クリップ一覧ビューを追加（Phase 2.5）**。全動画の区間を「独立したクリップ」として横断一覧できる。
  上部バーの「ライブラリ ⟷ クリップ」タブで表示を切り替える。
  - クリップカード: in 点サムネイル（`.dcm/thumbnails/` に自動生成・キャッシュ）/ ラベル（インライン編集可）/
    元動画名 / in–out・長さ / コーデック・解像度・fps バッジ。
  - カードをクリックすると元動画を開いて in 点へジャンプ（Ctrl+クリックで選択トグル）。カードから削除も可能。
  - 絞り込み（元動画）/ 並び替え（動画順・作成が新しい順・長い順・ラベル順）/ ラベル・動画名の検索。
  - **複数選択 → 動画横断のまとめて書き出し**に対応（書き出しモーダルをクリップ選択ベースに拡張。
    複数動画に跨るときは元動画名も表示）。
- クリップから開いたときの mpv 再生開始位置指定に対応（in 点から一時停止状態で表示）。

### 2026-07-08 15:10
- **HEVC/10bit を原本のまま再生できるよう mpv 埋め込み再生を追加**。動画領域にネイティブの mpv を埋め込み、
  ハードウェアデコードで無劣化・キャッシュ不要・即シークで再生する。再生/一時停止/シークはタイムラインと連動。
- mpv が見つからない/起動できない環境では、従来の再生（+ 必要時のみ一時プロキシ）に自動で切り替わる。
- 開発時は winget の mpv（`shinchiro.mpv`）を利用。配布時のアプリ同梱は今後対応。
- 埋め込み mpv が真っ暗になる問題を修正（Chromium の GPU コンポジタ無効化）。実機で HEVC 10bit 4K の再生を確認。

### 2026-07-08 14:30
- HEVC / 10bit のプレビューを **原本の直接再生を優先**する方式に変更。Chromium の platform HEVC デコーダを有効化した
  （Windows は「HEVC ビデオ拡張機能」があれば原本のまま再生可）。
- 直接再生できない環境向けに、プレイヤー上のボタンから **一時的に H.264 プロキシへ変換して再生** できるようにした。
  プロキシは **永続保存せず OS 一時フォルダに置き、アプリ終了時に自動削除**（容量を消費しない）。書き出しは常に元素材で無劣化。

### 2026-07-08 14:10
- （内部）HEVC/10bit プレビュー用に H.264 プロキシ生成を追加（この後 14:30 の変更で一時プロキシ方式へ改めた）。

### 2026-07-08 13:45
- **ロスレス書き出し（stream copy）を追加**。区間をキーフレーム整合のまま無劣化で個別クリップに書き出す。
  - 書き出しモーダル: 出力先フォルダ選択、命名テンプレート（`{filename}` / `{label}` / `{index}`）、区間の選択、逐次バッチ書き出しと進捗表示、成功/失敗サマリ。
  - DJI 素材のデータストリーム（timecode/telemetry）を自動的に除外して確実に書き出す。
- `start.bat` がダブルクリックで起動しない不具合を修正（改行コードとバッチ構造、Electron 起動時のクラッシュ回避）。

### 2026-07-08 13:10
- プロジェクト名を `drone-clip-manager` に統一（旧仮称 `flight-cut`）。ウィンドウタイトル / package.json / ドキュメントを更新。メタデータディレクトリは `.dcm/`。
- BGM プレイヤーを追加: BGM フォルダをルートとは独立に指定し、mp3 等を一覧・再生（前後送り / 音量 / リストループ）。
- 起動用 `start.bat` を追加（ダブルクリックで、必要なら `npm install` してから開発起動）。

### 2026-07-08 12:52
- アプリのスキャフォールドを追加（Electron + React + TypeScript / electron-vite）。
- ルートフォルダを指定してライブラリを取り込む機能を追加（ルート直下 `.dcm/` にメタデータを作成）。
- フォルダ階層ツリー、動画プレビュー再生、動画メタデータ表示（codec / 解像度 / fps / bit深度 / 色 / サイズ）を追加。
- タイムライン UI を追加: キーフレームマーカー表示、ドラッグで区間作成、再生ヘッド、区間バーの可視化。
- 区間ブックマークの作成 / ラベル編集 / 削除と、SQLite（`.dcm/library.db`）への永続化。
- in/out のキーフレームスナップ（in=直前 / out=直後）。`I` / `O` キーでの区間作成、`Space` で再生切替。
