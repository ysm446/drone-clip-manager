## 未リリース

### 2026-07-09 08:25

  そこでクリック/ドラッグすると動画だけをシークできるようにした（区間の作成・編集は下のトラックのまま）。

### 2026-07-09 08:05
- **クリップにユーザー定義タグを付けられるようにした（Phase 2.8）**。地名・時間帯などを自由記述で複数付与できる。
  - クリップカードにタグチップを表示。「＋タグ」入力で追加（既存タグを補完候補に表示）、× で個別に削除。
  - クリップビューのツールバー下に**タグ絞り込み**（複数選択は AND）。検索もタグを対象に含む。
  - タグは `.dcm/library.db` に保存（区間を削除するとタグも一緒に削除）。
  - ※ 今回はクリップ（区間）向け・自由タグ。動画へのタグ / カテゴリ / 時間帯の自動サジェストは今後。

### 2026-07-09 07:20
- **シーケンス（ノードグラフ）編集を追加（Phase 2.6・一本道 / 再生まで）**。上部バーに「シーケンス」タブを追加。
  - 左ペインでシーケンスを作成・選択・リネーム・削除。クリップパレットからノードとしてキャンバスに追加。
  - ノードはドラッグで配置。出力ポート → 入力ポートのドラッグで接続して 1 本の順路を組む（各ノードの
    入出力は 1 本＝一本道。閉路は不可）。エッジはクリックで削除、ノードは✕で削除。
  - ツールバーの **▶ 再生** で、組んだ順路のクリップを上部プレイヤーで先頭から連続再生（再生中ノードを強調）。
  - シーケンスはルート直下 `.dcm/library.db` に永続化（次回起動時も復元）。
  - ※ 無劣化の連結書き出し（ffmpeg concat）と分岐ノードは次の反復で対応予定。
- **再生画面をクリックで再生 / 一時停止**できるようにした（mpv 埋め込み・`<video>` 両方）。

### 2026-07-09 01:43

  - **F9: 現在の動画フレーム**をフル解像度で保存（mpv 再生中は mpv の現フレーム、それ以外は ffmpeg 抽出）

  併せて既定の DevTools ショートカット(F12)が無くなり、F12 をスクリーンショットに割り当てられる。

### 2026-07-09 01:01
- ウィンドウのコンテンツ領域（描画領域）を **1920×1080** にした（フレームを除いた内寸で指定）。
- **クリップビューでも上部にプレイヤーを常時表示**するようにレイアウトを変更。クリップ一覧はプレイヤーの下に配置。
- **クリップをクリックすると、ビューを切り替えずにその動画を上部プレイヤーで再生**（in 点へシーク）できるようにした。

### 2026-07-09 00:47
- **しばらく使う（特にシークを繰り返す）と動画が応答しなくなる問題を修正**。
  - 原因: mpv の stdout/stderr をパイプに繋いだまま誰も読んでいなかったため、ログが OS の
    パイプバッファ（約64KB）を埋めた時点で mpv の書き込みがブロックし、プレイヤー全体が停止していた。
  - 修正: `--no-terminal` でターミナル出力を停止し、stdio を切り離した。
- **mpv が落ちても自動復帰するようにした**。mpv プロセス / IPC の異常を検知したら mpv を再起動し、
  同じ動画を同じ再生位置から自動で開き直す（短時間に連続で落ちる場合は `<video>` 再生へフォールバック）。
  従来は一度異常が起きるとアプリを再起動するまで映像を制御できなくなっていた。

### 2026-07-08 23:54

  - どちらのモードでも既存区間の端リサイズ / 本体移動は可能。
- クリップサムネイルと mpv 埋め込みウィンドウの角丸を削除（Windows 11 既定の角丸で動画の角が欠けていた）。

### 2026-07-08 23:35
- **クリップ一覧ビューを追加（Phase 2.5）**。全動画の区間を「独立したクリップ」として横断一覧できる。
  上部バーの「ライブラリ ⟷ クリップ」タブで表示を切り替える。

  - カードをクリックすると元動画を開いて in 点へジャンプ（Ctrl+クリックで選択トグル）。カードから削除も可能。
  - 絞り込み（元動画）/ 並び替え（動画順・作成が新しい順・長い順・ラベル順）/ ラベル・動画名の検索。
  - **複数選択 → 動画横断のまとめて書き出し**に対応（書き出しモーダルをクリップ選択ベースに拡張。

- クリップから開いたときの mpv 再生開始位置指定に対応（in 点から一時停止状態で表示）。

### 2026-07-08 15:10
- **HEVC/10bit を原本のまま再生できるよう mpv 埋め込み再生を追加**。動画領域にネイティブの mpv を埋め込み、
  ハードウェアデコードで無劣化・キャッシュ不要・即シークで再生する。再生/一時停止/シークはタイムラインと連動。
- mpv が見つからない/起動できない環境では、従来の再生（+ 必要時のみ一時プロキシ）に自動で切り替わる。
- 開発時は winget の mpv（`shinchiro.mpv`）を利用。配布時のアプリ同梱は今後対応。
- 埋め込み mpv が真っ暗になる問題を修正（Chromium の GPU コンポジタ無効化）。実機で HEVC 10bit 4K の再生を確認。

### 2026-07-08 14:30
- HEVC / 10bit のプレビューを **原本の直接再生を優先**する方式に変更。Chromium の platform HEVC デコーダを有効化した
  （Windows は「HEVC ビデオ拡張機能」があれば原本のまま再生可）。
- 直接再生できない環境向けに、プレイヤー上のボタンから **一時的に H.264 プロキシへ変換して再生** できるようにした。
  プロキシは **永続保存せず OS 一時フォルダに置き、アプリ終了時に自動削除**（容量を消費しない）。書き出しは常に元素材で無劣化。

### 2026-07-08 14:10
